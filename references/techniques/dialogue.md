# 05 — 对话设计 (v1.0)

> 好对话像打乒乓球——有来有回，有旋转，有扣杀。

---

## 一、对话的核心法则

### 每句对话至少完成一项功能

| 功能 | 说明 | 示例 |
|------|------|------|
| **推动情节** | 传递新信息、改变局面 | "监控录像找到了。" |
| **揭示人物** | 暴露性格、态度、价值观 | "我不信任警察。他们从来没帮过我。" |
| **制造冲突** | 制造对立、激化矛盾 | "你骗了我。你从头到尾都在骗我。" |
| **传递信息** | 告诉读者需要知道的 | "会议改到明天下午三点。" |
| **表达情感** | 流露情绪状态 | "我……我不知道该说什么。" |
| **建立悬念** | 让读者想知道更多 | "那天晚上发生的事，你最好永远别问。" |

**无效对话就是应该删除的对话。**

### 真实的人不会完整表达

人有防御机制、有羞耻心、有不可告人的秘密。**对话的表面意思和真实意思往往不同。**

```
表面："我没事。"
真实："我有事但我不能说/不想说/说了你也不懂。"

表面："你走吧。"
真实："求你留下来，但我不能开口求你。"

表面："随便你。"
真实："我很在乎，但我已经累了。"
```

## 二、潜台词（Subtext）

### 核心概念

好对话的真正含义在**表面之下**。读者听到的≠人物真正想说的。

### 潜台词的四种实现方式

**1. 避而不谈**
> "你觉得他怎么样？"
> "咖啡凉了。"
> （评价一个人，用咖啡回避 → 不愿评价）

**2. 过度反应**
> "你跟王总聊了什么？"
> "没什么！就是工作上的事！"（音量不自觉地提高）
> （过度反应 = 此地无银）

**3. 答非所问**
> "你喜欢他吗？"
> "他做饭挺好吃的。"
> （回避正面问题 → 答案其实已经有了）

**4. 沉默即回答**
> "你是不是还爱着他？"
> 长久的沉默。
> 她起身去倒水。
> （不回答本身就是回答）

### 巧思：读者比说话者知道得多的对话

```
读者知道A在撒谎，但B不知道 → B相信了A
读者替B着急 → 强烈的阅读驱动力

读者知道B是A失散多年的弟弟，但A和B都不知道
两人对话中无意间暴露线索 → 读者又揪心又兴奋
```

## 三、角色辨识度：谁在说话

### 不需要"XX说"就能辨识说话人

如果你的对话去掉"他说""她说"之后读者分不清是谁在说话，那对话设计就是失败的。

### 五个辨识度维度

| 维度 | 差异化方法 |
|------|-----------|
| **词汇选择** | 不同教育背景用不同词汇 |
| **句长习惯** | 有人话多有人话少 |
| **口头禅** | 一两个就好，多了烦人 |
| **思维方式** | 有人直奔主题，有人绕半天 |
| **节奏感** | 有人说话像机关枪，有人慢条斯理 |

### 实战：同一场景，不同人物的表达

**场景：咖啡洒在键盘上了**

```
急性子："操！完了完了完了！"
慢性子："……哦。没关系，我去拿纸巾。"
毒舌："老天爷都在帮你决定今天不加班。"
强迫症："你等一下。先把电源拔了。别擦，先拔电源！我说了先拔电源！"
胆小鬼："完了，老板会杀了我的，怎么办怎么办……"
```

## 四、对话推动情节

### 对话不应该只是"聊天"

每段对话都要改变点什么——局面、关系、信息、情绪。

### 对话推动情节的四种方式

**1. 信息揭示**
> "监控拍到那天晚上有第三个人在场。"
> （情节立即转向新方向）

**2. 关系变化**
> "从今天起，你不是我哥。"
> （人物关系不可逆转地改变）

**3. 制造决策**
> "如果你今天不去，就永远别想再见她。"  
> 李明站起来。"车钥匙给我。"
> （对话导致行动）

**4. 暗示/铺垫**
> "你确定要住那个房间？"
> "怎么了？"
> "没什么。（顿了一下）晚上别开窗。"
> （不是直接信息揭露，但制造了悬念）

## 五、对话的节奏与停顿

### 停顿的力量

有时候**不说的比说的更有力**。

> "他问过我一个问题。"
> "什么问题？"
> "……"
> "什么问题？"
> "他问我，这辈子有没有做过让自己后悔的事。"
> "你怎么说？"
> "我没回答。到现在也没回答。"
> 
> （停顿 = 无言的情感重量）

### 巧思：对话+动作=沉默也能传达

```
"你真的要走？"
她继续叠衣服。一件。两件。三件。每一件都叠得整整齐齐。
"你说话啊。"
她把叠好的衣服放进行李箱。拉链拉上的声音，比任何回答都响。
```

## 六、对话常见败笔

| 败笔 | 表现 | 修复 |
|------|------|------|
| **信息倾倒** | 角色说读者需要知道的信息，但不自然 | 把信息拆散、混在动作里、通过冲突揭示 |
| **念稿式** | 无停顿、无打断、无迟疑 | 真人说话永远支离破碎 |
| **同质化** | 所有角色说话方式一样 | 给每个人不同的语言指纹 |
| **过度修饰** | "他愤怒地说""她高兴地回答道" | 对话内容本身应该传达情绪 |
| **无目的闲聊** | 没有任何功能的对话 | 删。哪怕写得再好也删。 |

## 对话设计自检

```
□ 每段对话至少完成一项功能？
□ 去掉"他说/她说"后能否辨识说话人？
□ 对话是否有潜台词层面？
□ 关键对话后，局面是否发生了改变？
□ 对话节奏是否有快有慢？
□ 是否有省略不当的"信息倾倒"？
□ 不同角色的词汇、句长、节奏是否有差异？
□ 是否有适时的停顿（沉默也是一种对话）？
```
