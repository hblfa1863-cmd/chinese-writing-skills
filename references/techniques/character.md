# 01 — 人物塑造 (v1.0)

> 读者记不住情节，但记得住人。人物是小说的灵魂。

---

## 一、人物弧光：角色的内在旅程

### 什么是人物弧光

人物在故事中发生的**根本性内在变化**——他开始时是谁，结束时变成了谁。

### 三种经典弧光

| 类型 | 描述 | 经典例子 |
|:---:|------|----------|
| **成长弧** | 从弱小/幼稚→强大/成熟 | 大多数成长小说主角 |
| **堕落弧** | 从善良/高尚→黑暗/邪恶 | 《绝命毒师》老白 |
| **平弧** | 人物不变，但改变世界 | 福尔摩斯、007 |

### 弧光设计三问

1. 故事开始时，主角**缺少什么**？（不是物质，是内在特质——勇气、信任、爱的能力）
2. 故事中，什么事件**迫使**他面对这个缺失？
3. 故事结束时，他**如何改变**？（是获得、丧失、还是接受？）

### 巧思：细微弧光

大的弧光跨越整本书，但每章可以有细微弧光：
- 第3章：傲慢 → 第一次被打击
- 第7章：开始怀疑自己
- 第12章：彻底放弃旧信念
- 第18章：重建新信念

## 二、矛盾构建：真实人物的核心

### 核心洞察

**真实的人充满矛盾。** AI 写的人之所以假，就是因为太"一致"了。

### 矛盾类型

| 类型 | 示例 |
|------|------|
| **身份矛盾** | 黑帮老大每天给孤儿院捐款 |
| **言行矛盾** | 满口仁义道德，私下蝇营狗苟 |
| **情感矛盾** | 恨一个人入骨，却又忍不住关心他 |
| **价值矛盾** | 想要自由，又害怕自由带来的责任 |
| **习惯矛盾** | 雷厉风行的女强人，回家对着亡夫照片说话 |
| **认知矛盾** | 最不信命的人，抽屉里藏着一张护身符 |

### 矛盾的使用原则

1. 矛盾必须与人物核心相关（不要为矛盾而矛盾）
2. 矛盾应该随情节推进而**演化**（不是静态设定）
3. 矛盾的**暴露时机**很重要——平时藏着，关键时刻显露

### 巧思：让读者自己发现矛盾

```
❌ 直接陈述：李明是一个冷酷的杀手，但他也有温柔的一面
✅ 侧面呈现：
  第5章：李明一枪毙了目标，面无表情地擦枪
  第7章：路过流浪猫，他蹲下来，放了一盒打开的罐头
  读者自己发现："这个人不简单"
```

## 三、侧面揭示技法

### 核心原则：展示 > 讲述

```
❌ 他非常愤怒。
✅ 他捏碎手中的纸杯，碎片扎进掌心也没松手。

❌ 她很紧张。  
✅ 她第三次调整了眼镜的位置。其实没歪。

❌ 他是个傲慢的人。
✅ 他从不直视下属的眼睛。说话的时候，视线永远越过对方的肩膀。
```

### 四种侧面揭示方法

#### 1. 用动作写心理

人的身体比嘴巴诚实。内心状态必然外化。

```
愤怒 → 捏碎东西、指关节发白、下颌紧绷
紧张 → 反复调整衣物、频繁看表、吞咽、舔嘴唇
恐惧 → 瞳孔放大、手心出汗、声音变调、本能后退
喜悦 → 脚步变轻、语调上扬、不自觉地笑、眼神发亮
悲伤 → 动作变慢、反应迟钝、食欲丧失、不愿说话
```

#### 2. 用选择写价值观

人在压力下的选择，比任何描述都有说服力。

> 火灾现场，他可以救名画，也可以救那只猫。他抱起了猫。
> ——不需要写"他善良"。

#### 3. 用习惯写秘密

反复出现的小习惯，是人物无意识的自曝。

- 紧张时摸无名指（曾经戴过戒指？后来摘了。）
- 永远坐背靠墙的位置（缺乏安全感？还是职业习惯？）
- 对每个服务员说谢谢（发自内心？还是某种补偿心理？）

#### 4. 用他人反应写地位

一个人是什么样的人，看别人怎么对他。

> 他一走进会议室，所有人突然安静了。
> ——不需要写"他很有威严"。

### 巧思：反面印证

有时候最有效的刻画是说反话：

> 所有人都说他是个好人。葬礼上来了一百多号人。
> 但没有一个人能说出一件他做过的好事。
> ——"好人"这个词突然有了讽刺意味。

## 四、人物出场设计

### 首次出场必须做的事

| 目标 | 方法 |
|------|------|
| **建立记忆点** | 一个独特的细节（动作/外貌/话语） |
| **暗示核心特质** | 通过行为暗示，不直接陈述 |
| **制造好奇心** | 让读者想知道更多 |

### 十种有效出场方式

**1. 行动中出场**（展示能力/性格）
> 李明跨过警戒线，警官试图拦住他。"市刑警队。"他亮出证件，径直走向尸体。

**2. 冲突中出场**（立即建立对立）
> "你迟到了。"她看了一眼手表，"四十分钟。"他拉开椅子坐下，没道歉。

**3. 反常中出场**（激发好奇）
> 她蹲在便利店门口吃关东煮，脚边放着爱马仕的袋子。

**4. 对话中出场**（先闻其声）
> "这种案子也拿来给我看？"声音从办公室最里面传来。然后李明才看到人。

**5. 物品中出场**（通过场景暗示）
> 办公桌上摆着三个相框，全部朝下扣着。

**6. 他人口中出场**（先入为主的印象）
> "千万别惹林姐。"新人入职第一天，同事只交代了这一句。

**7. 对比中出场**（环境反衬）
> 她穿着洗得发白的T恤走进米其林三星。比所有人都自在。

**8. 省略中出场**（欲说还休）
> 履历表上，2016年到2019年是空白的。面试官问起，他说："那三年我不存在。"

**9. 错误中出场**（暴露缺陷）
> 第一天上班，他把咖啡洒在了总监的键盘上。

**10. 极致状态下出场**（压迫感）
> 他已经三天没睡了。当门铃响起时，他几乎把杯子摔在地上。

### 出场禁忌

- ❌ 镜子自照式外貌描写
- ❌ 简历式自我介绍
- ❌ 无关紧要的日常活动（起床、刷牙、吃早餐）
- ❌ "这是一个怎样怎样的人"的直接陈述

## 五、反派深度

### 好反派的标准

1. **有自己的逻辑** — 他相信自己在做正确的事
2. **与主角有深层联系** — 不是单纯为了作恶而存在
3. **对主角产生实质威胁** — 不可信的威胁等于没威胁
4. **揭示主题** — 他代表了主角内心某个面向的极端化

### 反派动机层级

| 层级 | 动机 | 效果 |
|:---:|------|------|
| **浅层** | "我要统治世界" | 乏味、不真实 |
| **中层** | "我被世界伤害，所以报复" | 可理解但单薄 |
| **深层** | "我的女儿死于一种可治愈的病，但药厂为了利润延迟了上市。现在我要让这个系统付出代价" | 读者虽不认同，但能理解 |

### 深层洞察：反派是主角的黑暗镜像

最好的反派是主角**如果走了另一条路，就可能变成的人**。

```
主角：失去家人后 → 选择保护其他人
反派：失去家人后 → 选择毁灭一切

同一起点，不同选择 → 这才是真正让读者不寒而栗的反派
因为在某个瞬间，读者会想："如果是我，我会怎么选？"
```

### 给反派一个"对的"时刻

让反派在某件事上说对了。这会极大地增加复杂度：

> "你说我残忍？你为了保护你的家人杀了多少人？至少我不虚伪。"
> ——主角没法反驳。

## 六、人物成长量化检查

每次人物做出重要决定后，检查：

```
□ 这个决定符合他目前的成长阶段吗？
□ 如果不符合作早期性格，有合理解释吗？
□ 他的缺陷是否在关键时刻导致了失败？
□ 读者是否能感受到他的变化（不是被告知，是看到）？
□ 这一章，他的哪个侧面被首次展示？
```
