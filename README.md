# 中文小说写作技法大全

> 系统化的中文小说写作技巧知识库 · 模块化 · 可扩展

## 这是什么

一个面向 AI 辅助创作和人类写作者的中文小说写作技法知识库。包含12个核心模块，覆盖从人物塑造到结尾收束的完整写作链路。

## 快速开始

### 作为 OpenClaw Skill 使用

```bash
# 将整个目录复制到 skills 目录
cp -r chinese-writing-skills ~/.openclaw/workspace/skills/
```

使用时对 AI 说：
- "给我看看人物塑造的技巧"
- "怎么写好网文的爽点？"
- "帮我检查这一章的悬念设计"

### 作为独立参考资源

直接浏览 `references/techniques/` 下的各模块文件，每个模块独立成章。

## 如何贡献新模块

1. 在 `references/techniques/` 下创建新 `.md` 文件
2. 按照现有模块的格式编写（标题→章节→实战示例→自检清单→扩展说明）
3. 在 `SKILL.md` 的模块索引表中添加条目
4. 提交 PR 或直接更新

每个模块文件都是独立自包含的——新增模块不影响现有模块。

## 设计理念

- **模块化**：每个技法文件独立使用，不相互依赖
- **实战导向**：每个概念都有正反对比示例
- **持续迭代**：每个文件末尾标注版本号和可扩展方向
- **双重用途**：同时对 AI agent 和人类写作者友好
